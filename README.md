## Vencord Dev Manager
### A GUI for installing, reinstalling, uninstalling the dev version of Vencord and installing third-party plugins/themes.
![Vencord Dev Manager](img/main.png)
## Dependencies:
- [Python 3.12](https://www.python.org/downloads/)
- [CustomTkinter 5.3](https://pypi.org/project/custom2kinter/)
- [Git](https://git-scm.com/downloads)
- [Node.js](https://nodejs.org/en/download)
